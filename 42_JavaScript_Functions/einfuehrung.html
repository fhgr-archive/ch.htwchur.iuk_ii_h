Einfaches Ausführungsmodell

  • Umgebungen (Environments) sind Abbildungen von Bezeichnern zu Werten. Eine Umgebung kann auf eine äussere Umgebung (outer environment) verweisen.
  • Jede Funktion hat eine Umgebung.
  • Jede Funktion wird in einem Ausführungskontext (execution context) ausgeführt. Der Ausführungskontext wird durch die Umgebung bestimmt.
  • Eine closure bezeichnet eine Funktion zusammen mit ihrer Umgebung.
