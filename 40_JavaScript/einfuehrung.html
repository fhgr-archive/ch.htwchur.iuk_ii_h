Lernziele

Die Studierenden

  • können Programme in der Programmiersprache JavaScript entwickeln.

  • Nutzung von JavaScript in Webanwendungen
    • Manipulation des DOM-Baumes
    • Event-Handling

    (z.B. JQuery/JQuery Mobile)
