Lerninhalte

Objekte

  • Objekte haben Eigenschaften.
  • Auf Eigenschaften (engl. Properties) kann über die Punkt-Notation (dot notation) oder die Klammer-Notation (bracket notation) zugegriffen werden.
  • Objekte können über einen Objekt-Initialiser-Ausdruck (object initialiser expression) mittels geschweiften Klammern erzeugt werden.
  • Der Gleichheits-Operator (equals operator) prüft für Objekte auf Identität.
  • Mit dem Operator in kann die Existenz einer Eigenschaft geprüft werden. Mit dem Operator delete kann eine Eigenschaft entfernt werden.
  • Objekte werden oft mit Konstruktoren erzeugt.

Lerninhalte

Arrays

  • Arrays sind spezielle Objekte.
  • Arrays können mittels Array-Initialisierern (array initialiser) mit eckigen Klammern erzeugt werden.
  • Die Eigenschaft length gibt die Länge des Arrays zurück. Arrays haben Methoden push, pop, join.

Lerninhalte

Date

  • Der Konstruktor Date kann verwendet werden, um Date-Objekte zu erzeugen.

Argument-Objekt

  • Beim Ausführen einer Funktion wird ein Argument-Objekt erzeugt.
